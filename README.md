GUI для сервера измерения темепературы [сервер](https://github.com/ky0ufu/OC_lab5) \
Для gui использовался qt6. Необходимо установить зависимости (qt)
### IMPORTANT: Windows
Для установки и использования QT использовался `MSYS2`

### Сборка и запуск
Сборка Windows
```bat
.\win_build.bat
```
!Использует абсолютный путь к msys2 \
Запуск Windows
```bat
.\win_start.bat
```
! Использует абсолютный путь к msys2

---
Сборка UNIX
```sh
./build.sh
```
